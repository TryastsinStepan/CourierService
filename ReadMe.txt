Этот потрясающий проект наглядно демонстрирует работу курьерской службы. Цель проекта: из входных данных о курьерах и заказах составить оптимальное расписание для успешной работы и получения прибыли для компании. Программа предназначена для использования на персональных устройствах. Расписание - порядок выполнения курьером определённых заказов.
